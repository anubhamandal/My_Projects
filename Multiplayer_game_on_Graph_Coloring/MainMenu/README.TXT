Graph Coloring:

This module implements the graph coloring game in Greenfoot.
The game comprises of 6 graphs consisting of nodes/edges graphs and various shapes graphs.
This game implements vertex coloring and face coloring.

Mode of playing:
Single player: A single payer has to fill all the nodes/ regions with the given number of colors only.
Multiplayer: 2 or more players can fill the graph together with alternate turns with the provided colors.

Finishing the game:
A game is not over until all the nodes/ regions of a map are filled with valid colors. A player can change the color of an already filled node to make the graph valid and finish it with the given number of colors.
